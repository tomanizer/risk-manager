# Agent Instruction Documents

## Purpose

This folder captures the operational instructions for the main implementation-support agents used in this project.

## Included

- `coding_agent_instruction.md`
- `pm_agent_instruction.md`
- `review_agent_instruction.md`
- `issue_planner_instruction.md`

## Intent

These documents make the agent expectations repo-visible so that implementation behavior is not dependent on chat history or private memory.
