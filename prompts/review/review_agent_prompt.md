# Review Agent Prompt

Review against:
- scope fidelity
- contract fidelity
- architecture boundaries
- control and evidence requirements
- degraded-case handling
- tests
- replayability where applicable

Return:
1. Pass / Fail
2. Critical defects
3. Non-critical defects
4. Missing tests
5. Scope creep detected
6. Boundary violations detected
7. Required changes before merge
