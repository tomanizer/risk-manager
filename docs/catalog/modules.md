# Module Catalog

## Risk Analytics
Deterministic source for VaR, ES, shock catalogs, shock lineage, P&L, contributors, concentration, tail, and related summary services.

## FRTB / PLA Controls
Deterministic source for PLA, backtesting, deterioration flags, and desk control status.

## Limits & Approvals
Deterministic source for limits, utilization, breaches, waivers, and approval routing.

## Controls & Production Integrity
Deterministic source for freshness, completeness, lineage, reconciliation, publication readiness, and false-signal assessments.

## Governance & Reporting
Deterministic source for report artifacts, action registers, evidence bundles, challenge logs, and sign-off support.

## Capital & Desk Status
Deterministic source for watchlist state, desk posture, remediation priority, and consequence views.

## Model Inventory / Usage Registry
Lightweight registry for model metadata, intended use, limitations, boundaries, versions, and open governance issues.
