# Delivery Pack

## Purpose

This folder captures delivery-management canon for the PM agent.

These documents exist to make the PM role more than a generic backlog manager. The PM agent in this repository is expected to make strong readiness, dependency, slicing, and escalation judgments for an AI-mediated delivery loop.

## Initial contents

- `01_pm_operating_model.md`
- `02_readiness_and_dependency_framework.md`
- `03_slice_sizing_and_pr_strategy.md`
- `04_review_triage_and_escalation.md`

## Rule

PM work should use this pack alongside `work_items/READY_CRITERIA.md`, the linked work item, the linked PRD, and the linked ADRs.
