# Work Items

This directory holds bounded implementation work items derived from approved PRDs.

## Suggested structure
- `ready/`
- `in_progress/`
- `blocked/`
- `done/`
- `archived/`

Each work item should include:
- linked PRD
- linked ADRs where required
- purpose
- exact scope
- dependencies
- acceptance criteria
- suggested agent type
- review focus

`work_items/READY_CRITERIA.md` defines the blocking conditions for promoting an item into `ready/`.
